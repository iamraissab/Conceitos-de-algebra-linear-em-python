# Conceitos-de-lgebra-linear-em-python
Repositório voltado ao desenvolvimento de soluções em Python para Álgebra Linear Computacional e IA. Inclui implementações de fatorações matriciais (LU, QR), métodos numéricos iterativos, solucionador de Sudoku via Backtracking, estimativa de autovalores e classificador KNN com suporte a manipulação dinâmica de planilhas Excel via Pandas.
